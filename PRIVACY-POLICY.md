# Privacy Policy for IndustryConnect AI

**Last Updated:** February 19, 2026

## Overview
IndustryConnect AI is an educational demonstration application that provides interactive interfaces for two fictional businesses: Pizza Paradise and Fashion Factory. This app is designed to showcase MCP (Model Context Protocol) integration with ChatGPT for developers and educational purposes.

## Data Collection

### What We Collect
IndustryConnect AI collects minimal data necessary to provide its services:

**Pizza Paradise:**
- Pizza name and quantity for order demonstrations
- No payment information is collected (demo only)

**Fashion Factory:**
- Item name, size, and quantity for purchase demonstrations
- No payment information is collected (demo only)

### What We Don't Collect
- Payment card information (PCI DSS data)
- Protected health information (PHI)
- Government identifiers (SSN, passport numbers, etc.)
- Authentication credentials or API keys
- Location data
- Full chat history or conversation logs

## Data Usage

All data collected is used exclusively for:
1. Demonstrating the app's functionality
2. Responding to user requests within the ChatGPT interface
3. No data is stored permanently or used for analytics

## Data Storage

- **No persistent storage:** This is a demonstration app. No user data is stored in databases.
- **Session-only data:** Any data collected exists only during the active session and is not retained.
- **No third-party sharing:** We do not share, sell, or transfer user data to third parties.

## User Rights

Users have the right to:
- Use the app without providing personal information beyond what's necessary for demonstration
- Request information about data handling practices
- Stop using the app at any time
